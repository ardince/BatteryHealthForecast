# Contributing

Thank you for your interest in BatteryHealthForecast.

## Development Workflow

1. Create a feature branch.
2. Implement one feature.
3. Run the test suite.
4. Update documentation.
5. Submit a pull request.

## Coding Style

- Python 3.11+
- PEP 8
- Type hints
- Google-style docstrings
- Black formatting
- Ruff linting
- pytest

## Commit Convention

Examples:

feat(dataset): add BatteryDataset

feat(kalman): implement adaptive filter

fix(metrics): correct RMSE calculation

docs(readme): update installation guide
